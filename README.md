# Weather App API - Task 6

A modern weather application that fetches and displays real-time weather data using a weather API. Built with vanilla JavaScript and Vite for fast development.

## Features

- Real-time weather data fetching
- City search functionality
- Current temperature display
- Weather forecast cards
- Dynamic background based on weather conditions
- Responsive UI with smooth animations
- Error handling for failed API requests

## Tech Stack

- **Frontend:** Vanilla JavaScript (ES6+ Modules)
- **Build Tool:** Vite
- **Styling:** CSS
- **Fonts:** Google Fonts (EB Garamond, Quicksand)
- **Icons:** Google Material Symbols Rounded

## Project Structure

```
WeatherApp/
├── index.html
├── package.json
└── src/
    ├── main.js
    ├── style.css
    ├── assets/
    └── scripts/
        ├── fetchWeather.js   # API fetching logic & state management
        └── ui.js             # UI rendering functions
```

## Getting Started

### Prerequisites

- Node.js installed
- Bun or npm package manager

### Installation

```bash
cd WeatherApp
npm install
```

### Development

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Usage

1. Open the app in your browser
2. Search for a city to get weather information
3. View current temperature, weather details, and forecast cards
